## 🚀 Мои проекты

### 🎮 AI Coach для Counter-Strike 2
> **Full-stack веб-приложение для автоматического анализа игровой статистики пользователей с использованием искусственного интеллекта.**  
> 🔗 **Live Demo:** [cs-coach.ru](https://cs-coach.ru)

*   **Стек технологий:** React (Vite), TailwindCSS, Python (FastAPI), Groq API (LLM), Steam Web API, PostgreSQL.
*   **Что реализовано технически:**
    *   **Авторизация через сторонний сервис:** Интегрировал защищённый вход на сайт через протокол OpenID (Steam Auth) для мгновенного получения уникального игрового ID пользователя без регистрации.
    *   **Проектирование бэкенда:** Разработал быстрый асинхронный бэкенд на Python (FastAPI) для обработки кастомных запросов и агрегации игровых метрик.
    *   **Интеграция ИИ (LLM):** Настроил связку с Groq API и спроектировал систему промптов (Prompt Engineering) для того, чтобы нейросеть анализировала массив сухих игровых данных и превращала их в развёрнутые текстовые рекомендации.
    *   **Фронтенд на React:** Создал адаптивный пользовательский интерфейс с динамическим отображением статистики и результатов ИИ-генерации.

### 🤖 Fake News Detector Telegram Bot
> **Асинхронный Telegram-бот на базе искусственного интеллекта для анализа текстовых новостей и определения их достоверности.**  
> 🔗 **Ссылка на бота:** [@NewsCheccker_bot](https://t.me/NewsCheccker_bot)

*   **Стек технологий:** Python, Aiogram (v3), Groq API, Llama 3 (LLM модель).
*   **Что реализовано технически:**
    *   **Асинхронная архитектура:** Разработал бота на базе фреймворка `aiogram`, что позволяет обрабатывать сотни запросов от пользователей одновременно без задержек и зависаний.
    *   **Интеграция с передовыми LLM:** Настроил интеграцию с нейросетью Llama 3 через сверхбыстрое API от Groq, обеспечив моментальный текстовый анализ.
    *   **Промпт-инжиниринг для фактчекинга:** Спроектировал системные инструкции для ИИ, благодаря которым модель не просто угадывает «правда/ложь», а раскладывает новость по критериям манипуляции, кликбейта и логических ошибок.
    *   **Архитектурный финал:** Проект успешно защищён в качестве итоговой практической работы в колледже, подтвердив свою техническую зрелость.

### 🌐 Web3 Token Minting dApp
> **Децентрализованное приложение (dApp) для покупки кастомных токенов стандарта ERC-20 в тестовой сети блокчейна Ethereum.**  
> 🔗 **GitHub репозиторий:** [WEB3-Token-Minting-dApp](https://github.com/Edvart08/WEB3-Token-Minting-dApp)

*   **Стек технологий:** React (Vite), Wagmi, RainbowKit, Viem, Solidity, Remix IDE, MetaMask.
*   **Что реализовано технически:**
    *   **Разработка и деплой смарт-контракта:** Написал на Solidity и успешно разместил в тестовой сети Sepolia смарт-контракт именного токена с логикой автоматического обмена входящего ETH на токены проекта.
    *   **Интеграция с Web3-провайдерами:** Настроил безопасное подключение некастодиальных криптокошельков (MetaMask) к интерфейсу приложения с использованием библиотек RainbowKit и Wagmi.
    *   **Связка фронтенда с блокчейном:** Реализовал асинхронное чтение балансов токенов напрямую из смарт-контракта с помощью хуков `wagmi` и обработку транзакций отправки средств на контракт.
